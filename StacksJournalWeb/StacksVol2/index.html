How a genetic algorithm went from convergence collapse to a 35-ticker diversity breakthrough in 72 hours

Over four versions and three sleepless nights, the AGENT system — Agent Guided Evolution Neural Trainer — evolved from a textbook genetic algorithm into a hybrid framework that combines population-based search with LLM-guided interpretation. Inspired by Karpathy’s AutoResearch, it applies recursive self-improvement not to training language models, but to evolving boolean gate trees for a covered-call portfolio. This is the full story of AGENT’s four generations, the failures that forced each redesign, and the roadmap for V5.
